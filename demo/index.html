Chaperone.js

A simple jQuery plugin for doing guided tours through a website.

Default Settings

By default chaperone makes some simple assumptions, see below for demo of what the defaults get you:

Markup

              <ol id="demo1">
                <li data-target="#demo1a" data-title="First">
                  Step 1
                </li>
                <li data-target="#demo1b" data-title="Second" data-placement="bottom">
                  Step 2
                </li>
                <li data-target="#demo1c" data-title="Third">
                  Step 3
                </li>
              </ol>

              <h2 id="demo1a">A</h2>
              <h2 id="demo1b">B</h2>
              <h2 id="demo1c">C</h2>

Javascript

              $(function(){
                $('#demo1').chaperone();
                $('#demo1').chaperone('start');
              });
If you close the tour, you can start it again by clicking here.
